## About The Project

English Invader is a learning app using Space Invader gameplay to learn words in english. This project is a english scholar project.

## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

   ```sh
   git clone git@github.com:adrien-lrsse/english-invader.git
   ```

2. Build all

   ```sh
   docker compose up --build
   ```
